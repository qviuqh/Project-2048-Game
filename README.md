# Project-2048-Game

## Requirements:
The notebooks are developed with:
* Python 3
* Numpy
* Pygame (play in window game)
* Pynput (play in terminal)
* Math

## How to run game program:
* Play in Terminal: run file "Main.py" 
* Play in window game: run file "Display.py"
